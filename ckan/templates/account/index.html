Accounts

CKAN uses OpenID for user authentication thus you do not need to register a new username and password to use this site - instead you can reuse your existing openid account.

Getting an OpenID

If you haven't already got an OpenID it is easy to get one. Probably the easiest way is sign up with a free OpenID provider such as https://www.myopenid.com/.

Logging In

If you have already got an OpenID just visit the ${XML( h.link_to('login page', h.url_for(controller='account', action='login')) )} .
