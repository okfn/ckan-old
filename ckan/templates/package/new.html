Please choose a short name for the package (once this is done you will be able to edit the other package attributes).

The name has several restrictions because it is used as a universal identifier for the package:

  • cannot match the name of any other package
  • must be at least 2 characters in length
  • must be in lower case
  • can only contain alphanumeric characters, dashes and underscrores
  • cannot match one of the reserved names

Package names can be changed after registration.
